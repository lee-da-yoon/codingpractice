### 함수 calloc()
- 함수원형 `void * calloc (size_t num, size_t size);`
	
	원소 크기가 size인 배열 크기 num개의 공간을 할당하여 포인터를 반환하고 원소값은 모두 0으로 저장 
	- num - 할당될 메모리 항목의 개수
	- size - 할당될 메모리 한 원소의 크기
	```
	int *ary = NULL: 
	ary = (int *) calloc (3, sizeof(int))
	반환값은 이 값을 받는 자료유형의 포인터로 변환하여 포인터 변수에 저장된다. 
	```

	> malloc() - 메모리만 할당, 초기화 x -> 쓰레기값 

	> calloc() - 메모리 할당 + 0으로 초기화

	> realloc() - 기존 메모리 크기 변경, 새로 늘어난 부분은 초기화되지 않음. 

	```
	[slide 12]

	int *arr1 = (int *) malloc (5 * sizeof(int));
	int *arr2 = (int*) calloc(5, sizeof(int));

	for (int i = 0; i < 5; i++{
		printf("%d "", arr1[i]);
	}
	printf("\n");

	for (int i = 0; i < 5; i++){
		printf("%d ", arr2[i]);
	}
	printf("\n");

	free (arr1);
	free (arr2);  // 두개 결과는 같은데 하나는 malloc 써서 5칸짜리 배열 만들기 하나는 calloc 써서 만들기 
	```

### 함수 realloc()
- 함수원형 `void * realloc (void *p, size_t size); `
- 이미 확보한 메모리 p를 다시 지정한 크기 size로 변경하는 함수이며, 이미 확보한 p가 NULL이면 malloc()와 같은 기능을 수행.
