### 문자열의 입출력 
- 함수 scanf()는 공백으로 구분되는 하나의 문자열 입력
	- 입력 받을 문자열이 저장될 충분한 공간인 문자 배열 선언
	- `scanf("%s", str)` 이용하여 문자열 입력 
	- `printf("%s", str)` 로 문자열 출력 
- gets()와 puts()
	- `gets()` 한 행의 문자열 입력 -> **fgets 사용 권장**
		- 엔터키 누를 때 까지 한 행을 버퍼에 저장 후 입력처리 
		- `문자열\n\0` 으로 인자인 배열에 저장된다.
